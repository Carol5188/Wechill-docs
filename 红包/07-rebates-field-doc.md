# 红包返奖记录页面字段说明文档

版本日期：2026-04-20  
对应页面：`hongbao-admin-pages/07-rebates.html`  
适用对象：产品、后台研发、服务端研发、测试、运营、风控审核人员

## 1. 页面定位

红包返奖记录页面用于按返奖记录 ID 查看已领完红包的返奖结算情况。页面重点帮助运营确认：

- 哪些已领完红包进入返奖结算记录。
- 返奖金额、返奖档位和发放状态是否符合配置。
- 发送人、房间和红包 ID 是否可以快速追溯。
- 普通记录是否需要发起风控处理。
- 已风控记录是否需要调整为危险级、限制级或观察名单。

## 2. 关键口径

| 口径 | 说明 |
| --- | --- |
| 返奖记录 ID | 单条返奖结算记录的唯一标识。 |
| 红包 ID / 开奖主键 | 当前返奖记录关联的红包 ID。页面用于排查、客服追溯和风控跳转。 |
| 已领完 | 红包全部份额已被领取。当前页面的开奖条件列只展示已领完红包。 |
| 返奖档位 | 当前红包命中的返奖规则，示例为 A档和 B档。 |
| 返奖金额 | 按红包金额或礼物总价值计算出的返奖金币，向下取整。 |
| 发放状态 | 当前返奖金币是否需要发放以及是否已完成发放。 |
| 风控状态 | 当前返奖记录关联用户的风控处理状态，包括危险级、限制级、观察名单和正常。 |

## 3. 顶部数据总览

| 字段 | 含义 | 展示说明 |
| --- | --- | --- |
| 今日开奖红包 | 当日进入返奖结算记录的已领完红包数量。 | 当前展示 `967`。 |
| 今日返奖金额 | 当日返奖金币总额。 | 当前展示 `7,683`，副文案拆分 A档和 B档金额。 |
| 中奖红包数 | 当日产生返奖金额的红包数量。 | 按返奖记录 ID 去重统计。 |
| 风控拦截 | 当日命中风控状态的返奖记录数量。 | 用于运营快速查看风险规模。 |

## 4. 筛选区字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| 结算时间 | 时间范围输入 | 按返奖结算时间筛选。 |
| 返奖记录 ID | 输入框 | 精确查询单条返奖记录。 |
| 红包 ID | 输入框 | 按红包 ID 查询关联返奖记录。 |
| 发送人 UID | 输入框 | 按红包发送人查询返奖记录。 |
| 返奖档位 | 下拉 | 可选全部、A档、B档。 |
| 风控状态 | 下拉 | 可选全部、危险级：禁发禁领、限制级：不参与返奖、观察名单、正常。 |
| 重置 | 按钮 | 清空筛选条件并恢复默认查询口径。 |
| 查询 | 按钮 | 按当前筛选条件刷新列表和统计数据。 |
| 导出 | 按钮 | 导出当前筛选结果。 |

## 5. 返奖记录列表字段

| 字段 | 含义 | 展示样式/示例 | 研发备注 |
| --- | --- | --- | --- |
| 返奖记录 | 返奖结算记录 ID 和记录类型。 | `REBATE-20260416-0012`，副文本 `A档返奖`。 | 建议作为列表主键。 |
| 红包 ID / 开奖主键 | 当前返奖记录关联的红包 ID。 | `HB202604160012`，副文本 `开奖主键`。 | 风控跳转时需要携带。 |
| 发送人 / 房间 | 红包发送人 UID 和房间 ID。 | 主文本 `UID 983211`，副文本 `房间 880321`。 | 运营排查时用于关联用户和房间。 |
| 红包金额与档位 | 红包总金额或礼物折算金额，以及命中的返奖档位。 | `20,000 金币`，副文本 `A档：3%返5%`。 | 礼物红包按礼物总价值折算金币。 |
| 开奖条件 | 当前返奖记录进入页面展示的前置条件。 | 仅展示 `已领完`。 | 未领完红包不进入当前列表展示。 |
| 返奖金额 | 本次结算出的返奖金币。 | `1,000 金币`、`6,664 金币`、`0`。 | 金币值向下取整。 |
| 发放状态 | 当前返奖金币发放状态。 | `发放成功`、`无需发放`、`未发放`。 | 与返奖金额和风控状态联动。 |
| 风控状态 | 当前记录的风险状态。 | `正常`、`危险级：禁发禁领`、`限制级：不参与返奖`、`观察名单`。 | 已风控状态在操作列可继续调整。 |
| 结算时间 | 当前返奖记录的结算时间。 | `2026-04-16 10:10:03`。 | 建议后端返回秒级时间。 |
| 操作 | 当前记录可执行动作或风控状态。 | 普通记录展示 `风控`；已风控记录展示对应风控状态。 | 交互同红包领取记录页。 |

## 6. 操作与弹窗

| 操作/状态 | 交互 | 结果 |
| --- | --- | --- |
| 风控 | 点击后跳转红包风控明细页，并打开手动加黑名单弹窗。 | URL 携带红包 ID、返奖记录 ID、发送人 UID、房间 ID、来源为返奖记录人工复核。 |
| 危险级：禁发禁领 | 点击后打开调整风控等级弹窗。 | 可切换为危险级、限制级或观察名单。 |
| 限制级：不参与返奖 | 点击后打开调整风控等级弹窗。 | 可切换为危险级、限制级或观察名单。 |
| 观察名单 | 点击后打开调整风控等级弹窗。 | 可切换为危险级、限制级或观察名单。 |

### 6.1 调整风控等级弹窗

已进入风控状态的返奖记录，在操作列点击当前状态后打开“调整风控等级”弹窗。

| 字段/选项 | 说明 |
| --- | --- |
| 弹窗标题 | 固定展示“调整风控等级”。 |
| 弹窗副标题 | 展示当前返奖记录 ID 和发送人 UID。 |
| 危险级：禁发禁领 | 用户不可发红包、不可领取红包。 |
| 限制级：不参与返奖 | 用户可发可领，但发送的达标红包不参与高额红包概率返奖。 |
| 观察名单 | 只做监控和预警，不直接拦截。 |
| 取消 | 关闭弹窗，不修改当前风控状态。 |
| 选择任一等级 | 更新当前返奖记录的风控状态和操作列展示。 |

### 7.2 返奖结算链路

| 节点 | 含义 |
| --- | --- |
| 红包领完 | 红包全部份额被领取，满足本页展示条件。 |
| 档位确认 | 根据红包金额或礼物折算金额确认 A档或 B档。 |
| 金额计算 | 按返奖比例计算返奖金额，并向下取整。 |
| 发放完成 | 返奖金币写入发送人账户。 |

### 7.3 返奖口径

| 口径 | 说明 |
| --- | --- |
| 展示范围 | 本页仅展示已领完红包的返奖结算记录。 |
| A档 | 20,000 - 49,999 金币，3% 概率返 5%。 |
| B档 | 大于等于 50,000 金币，5% 概率返 10%。 |
| 礼物红包 | 按礼物总价值折算金币参与返奖，返奖发给发送人金币。 |
| 风控处理 | 命中危险级、限制级或观察名单时，运营可在操作列调整风控状态。 |

## 10. 测试验收点

- 顶部今日开奖红包数量应与列表中已领完返奖记录口径一致。
- 筛选区应展示风控状态筛选，不展示无关筛选项。
- 列表开奖条件列只展示已领完。
- 列表操作列应与红包领取记录页一致：普通记录展示风控，已风控记录展示危险级、限制级或观察名单。
- 点击风控，应跳转红包风控明细页并打开手动加黑弹窗。
- 点击已风控状态，应打开调整风控等级弹窗，并支持切换危险级：禁发禁领、限制级：不参与返奖和观察名单。
- 调整风控等级后，应同步更新当前行风控状态列和操作列。
