## 推荐阅读顺序

1. `幸运礼物玩法方案-共享奖池评审版.md`
   - 当前最新评审口径。

2. `README-三版对比.md`
   - A/B/C 三个方案的定位对比。

3. `幸运礼物ABC方案-流程图与流程表.md`
   - 评审会流程说明。
   - 包含通用入口流程和三版差异流程。

4. `幸运礼物玩法方案-A基础版（无奖池）.md`
   - 单次抽奖、无奖池。
   - 适合作为快速上线或日常底盘参考。

5. `幸运礼物玩法方案-B完整版（含奖池）.md`
   - 单次抽奖 + 房间独立奖池。
   - 保留为历史方案和对比参考。

6. `幸运礼物玩法方案-C共享奖池版（Greedy式）.md`
   - 活动/国家共享奖池方案。
