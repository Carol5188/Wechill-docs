# Wechill 公会薪资管理后台 PRD

> 文档版本：v1.0  
> 生成日期：2026-05-07  
> 产品：wechill  
> 模块：公会管理 - 主播薪资结算

---

## 一、需求背景

### 1.1 业务背景
公会体系是语聊房产品的核心运营机制，涉及公会长、管理员、主播三层角色。当前后台薪资结算页面字段不够直观，运营无法快速查看主播/公会的关键数据变化，影响结算效率和数据核对。

### 1.2 需求目标
1. **提高数据可读性**：新增钻石变化字段，直观展示筛选周期内的钻石增减
2. **优化筛选体验**：增加快捷时间选项，减少手动选择成本
3. **补全数据看板**：新增公会数据概览、主播数据概览、趋势分析等可视化能力

---

## 二、页面调整需求

### 2.1 日期精确度提高

**当前问题**：筛选时间只能手动选择起止日期，操作繁琐。

**优化方案**：
- 新增5个快捷选项：当天、本周、上周、本周期、上一周期
- 默认筛选时间段为当前结算周期

**UI样式参考**：
```
[当天] [本周] [上周] [本周期] [上一周期] | [2026-03-01] 至 [2026-03-15 23:59:59]
```

### 2.2 筛选条件调整

**现有筛选项**：
- 日期范围
- 用户ID
- 公会长ID
- 公会ID
- 公会地区（按大区筛选）

**保持不变**，无需新增筛选条件。

### 2.3 列表字段调整

**原字段**：
| 序号 | 字段名 | 说明 |
|------|--------|------|
| 1 | 用户ID | 主播用户ID |
| 2 | 用户昵称 | 主播昵称 |
| 3 | 用户头像 | 主播头像 |
| 4 | 公会ID | 所属公会ID |
| 5 | 所属公会ID | 公会父级ID |
| 6 | 公会地区 | 沙特/英国/中国/美国等 |
| 7 | 活跃天数(天) | 本周期活跃天数 |
| 8 | 在线时长(H) | 本周期在线总时长 |
| 9 | 本期总钻石 | 筛选周期内获得的钻石总数 |
| 10 | 本期剩余钻石 | 当前钻石余额 |
| 11 | 本期主播总薪资($) | 本周期应发工资 |
| 12 | 本期主播剩余薪资($) | 扣除预支后余额 |
| 13 | 本期主播1G奖励($) | TG奖励金额 |
| 14 | 预支金额 | 已预支金额 |

**新增字段**：

| 序号 | 字段名 | 计算逻辑 | 说明 |
|------|--------|----------|------|
| 15 | 期初钻石余额 | 筛选时间范围起点的钻石余额 | 用于计算变化量 |
| 16 | 期末钻石余额 | 筛选时间范围终点的钻石余额 | 用于计算变化量 |
| 17 | 钻石变化 | 期末钻石余额 - 期初钻石余额 | 正数为增加，负数为减少，标红/绿显示 |

**字段排序调整**：
```
用户ID → 用户昵称 → 用户头像 → 公会ID → 所属公会ID → 公会地区 
→ 活跃天数 → 在线时长 → 期初钻石余额 → 本期总钻石 → 期末钻石余额 
→ 钻石变化 → 本期剩余钻石 → 本期主播总薪资 → 本期主播剩余薪资 
→ 本期主播1G奖励 → 预支金额
```

## 五、数据来源与计算规则

### 5.1 钻石余额与流水定义
- **钻石余额**：用户钱包中通过礼物获得的钻石 + 运营后台赠送钻石
- **钻石流水**：本周期内获得的钻石总数（含普通礼物100%、幸运礼物10%）
- 不计入：充值获得钻石、系统赠送钻石、活动奖励钻石

### 5.2 工资计算规则

**双周政策**：
- 主播工资 = 钻石余额 ÷ 25000（钻石转美元比例）
- TG奖励 = 钻石流水 × 对应档位奖励比例
- 任务要求：每周期完成6天×2小时，未完成扣30%

**自由政策**：
- 主播工资 = 钻石余额 ÷ 16026 × 75%
- TG奖励 = 钻石流水 × 对应档位奖励比例
- 最低结算金额：$10

### 5.3 自赠礼物折算
- 普通礼物自赠：钻石 = 礼物金币 × 99%
- 幸运礼物自赠：钻石 = 礼物金币 × 10% × 99%

## 六、交互细节

### 6.1 筛选器交互
- 点击快捷时间按钮，自动填充起止日期
- 快捷按钮高亮当前选中状态
- 手动修改日期后，快捷按钮取消高亮
- 结算周期按双周政策计算（1-15日、16-月末）

### 6.2 列表交互
- 钻石变化字段：正数绿色↑、负数红色↓、0灰色-
- 支持按钻石变化排序（升序/降序）
- 支持导出Excel

### 6.3 看板交互
- 卡片支持点击查看明细列表
- 图表支持hover显示具体数值
- 排名榜支持点击跳转对应公会/主播详情页
